## How to use this repo
1. Clone the repo.
2. Create a virtual environment (recommended).
3. Install dependencies from requirements.txt:
   - pip install -r requirements.txt
4. Browse the Day-* folders — each contains notes.md, example code, and assets.
5. Run demos locally (see each day's README for instructions).

## Contributing
